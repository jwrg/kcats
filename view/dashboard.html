This app scaffold supports basic timestamped JSON data stored in PostgreSQL in a bespoke database and data table. The table is organized into three columns; one each for an ID, a timestamp, and the actual JSON data. In this way, the data is uniquely ID'ed and can be sorted easily by timestamp and visualized using a calendar view.
